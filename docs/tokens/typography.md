# typography.base

**Stability:** beta — public consumer API, usable but contract may still change.

Token tipografici.

## API

- `fontFamily` (`regular`, `mono`)
- `fontSize` (`xs`..`xxxl`)
- `lineHeight` (`tight`, `normal`, `relaxed`)
- `fontWeight` (`regular`, `medium`, `semibold`, `bold`)
- `typography` (aggregato)

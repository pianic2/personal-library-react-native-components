# Textarea

Wrapper di `Input` configurato come `multiline`.

## Import

```ts
import { Textarea } from "@personal-library/react-native-components";
```

## Props

Attualmente la funzione è tipizzata come `props: any` e le props vengono passate a `Input`.

## Esempio

```tsx
<Textarea label="Note" helperText="Max 500 caratteri" />
```

# PasswordInput

**Stability:** internal / non-stable — not part of the public consumer API.

Wrapper di `Input` che abilita/disabilita `secureTextEntry`.

## Import

```ts
import { PasswordInput } from "@personal-library/react-native-components";
```

## Props

Attualmente la funzione è tipizzata come `props: any` e le props vengono passate a `Input`.

## Esempio

```tsx
<PasswordInput label="Password" placeholder="••••••••" />
```
